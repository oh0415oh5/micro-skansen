## 4. 개발 환경

### 4.1 클라우드 VM 사전 설치 런타임

Cursor Cloud VM에는 아래 런타임이 기본 제공됩니다.

| 런타임 | 확인 명령 |
|--------|-----------|
| Node.js (nvm) | `node -v` |
| Python 3.12 | `python3 --version` |
| Go 1.22 | `go version` |
| Rust | `rustc --version` |

### 4.2 실행에 필요한 서비스

현재 실행할 서비스가 없습니다.

| 서비스 | 필요 여부 | 비고 |
|--------|-----------|------|
| 애플리케이션 서버 | 해당 없음 | 앱 미구현 |
| 데이터베이스 / 캐시 | 해당 없음 | 미설정 |
| Docker Compose | 해당 없음 | 설정 파일 없음 |

### 4.3 의존성 설치

`package.json`, `requirements.txt` 등 의존성 매니페스트가 없어 설치 단계는 **현재 no-op**입니다. 애플리케이션 코드가 추가되면 해당 스택에 맞는 설치 명령을 정의해야 합니다.
